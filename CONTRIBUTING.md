# Contributing to Antigravity

We welcome contributions to the Game Theory Lab!

## Tech Stack
- **Frontend**: React 19, TypeScript, Vite, Plotly.js
- **Backend**: Python 3.12, FastAPI, NumPy, SciPy

## How to Contribute

1. **Fork the repository**
2. **Setup Local Environment**:
   - Backend: `cd backend && python -m venv venv && source venv/bin/activate && pip install -r requirements.txt`
   - Frontend: `cd frontend && npm install`
3. **Create a Feature Branch**
4. **Implement your Simulation/Feature**
5. **Open a Pull Request**

## Coding Standards
- **Python**: Follow PEP 8.
- **TypeScript**: Use ESLint configuration provided.
- **Commits**: Use conventional commits (e.g., `feat: add nash equilibrium solver`).
